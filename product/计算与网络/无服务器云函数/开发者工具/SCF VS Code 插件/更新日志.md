 ## v1.1.14 版本 
 发布时间：2019-08-15
 
 本次发布包含了以下内容：

**新增功能：**
函数相关操作（创建，调试，上传）时，插件自动检测 SCF CLI 的安装状况。

**问题修复：**

- 云端函数同步至本地时，无权限等异常提醒完善。
- 插件自动安装 CLI 的错误信息输出。
- 云端函数同步至本地后默认打开工作区。
- 其他体验调优。

## v1.0.27 版本
发布时间：2019-07-11  

本次发布包含了以下内容：

**新增功能：**
- 支持云端函数导入本地时，自动加载 `template.yaml` ，可右键直接跳转编辑代码。
- 优化云端、本地、云端&本地状态展示。
- 其他体验优化。

- Python 函数本地调试异常。

- 其他已知问题修复。
